# 𓃥 白六雙擺律動裝置 (Double Pendulum Wave 3D Metallic Simulation)

一個結合 WebGL 3D 鏡面光澤視覺與 Web Audio API 空間立體聲的非線性混沌動力學（Chaos Theory）裝置。本專案精確模擬了多組雙擺（Double Pendulum）在重力作用下的非線性複雜軌道，並將其幾何外觀與聲學反饋提升至感官藝術層次，打造出一個極具催眠、深邃且療癒的數位「放空藝術裝置」。

## 🌟 項目特色

- **極致金屬光澤感（鏡面流光反射）**：採用 WebGL 自訂代碼即時動態生成一張**劇院級黑白幾何高對比環境貼圖（Generated EnvMap）**。當擺臂與圓盤進行大角度混沌翻轉時，拋光鏡面會絲滑地倒映出外界的線性流光，完美呈現「奢華鏡面金（Gold）」與「水銀太空銀（Silver）」的真實金屬重工業質感，徹底解決全黑背景下金屬死黑隱形的問題。
- **等比遞減長桿結構**：14 組雙擺的第一擺臂（長桿 $L_1$）從前端到後端呈現完美的等比例依序遞減設計。並可透過控制面板拉桿一律對所有長桿進行等比縮短壓縮，幾何線條極具張力。
- **刀片薄扁圓盤防阻礙設計**：擺臂末端採用極薄的扁平圓盤（Disc）取代傳統立體大球體，圓盤正面迎擊視線。在多組雙擺進行 360 度大翻轉、交叉穿梭的極限運動中，能像刀片般流暢穿梭，**絕不對任何運動軌跡造成視覺或幾何上的碰撞阻礙**。
- **聲學非線性律動（雙節最高點發聲）**：精確偵測第一節長桿與第二節短桿擺盪至左、右兩側最高處（即角速度 $\omega$ 穿過零、正負號反轉的反折點）的瞬間，即時觸發清脆的水晶音樂盒音效。音階分配採用**五聲音階（Pentatonic Scale）**，短桿發出的音頻比長桿更高、更清脆，隨著混沌運動編織出不重複的空靈樂章。
- **空間立體聲聲道（Stereo Panning）**：即時計算每個圓盤在 3D 場景中的 Z 軸空間位置，將聲音實時分配至左、右耳耳機聲道，配戴耳機觀賞時能獲得極佳的環繞立體聲放空體驗。
- **黃金俯視平視角**：預設相機視線微調至「剛好略高於雙擺盤組」的黃金側面切面，開場畫面飽滿，並預設自動收合操作面板。
- **智慧收合操作介面**：在介面開啟狀態下，只要點擊或觸碰 3D 顯示畫布區（Canvas），介面便會自動向左上角流暢滑動收合，畫面保持絕對純淨。

## ⚙️ 互動控制功能

操作面板提供以下即時調校參數：

1. **擺臂色彩主題**：
   - `純奢華金 (Metallic Gold)`：高拋光鏡面金，金屬度 1.0，極低粗糙度。
   - `純太空銀 (Metallic Silver)`：水銀亮銀質感，全面折射鏡面條形光。
   - `經典彩虹漸層`、`翡翠綠漸層`、`極光藍紫漸層`、`賽博霓虹漸層`。
2. **設定初始釋放角度**：可自由調整長桿（$\theta_1$）與短桿（$\theta_2$）的開局角度（20° 至 170°）。
3. **長桿縮短比例拉桿**：一律等比例將所有雙擺的長桿向下壓縮變短，完美確保短桿長度永遠在安全限制內。
4. **模擬演進速度拉桿**：內建高階 RK4 物理多步迭代器（0.2x 至 3.0x），可即時加快混沌運動的演進流速，聲音節奏也隨之同步成倍變化。
5. **動態控制按鈕**：提供「開始/停止」凍結功能與「重新釋放」物理狀態歸零鈕。

## 🔬 物理與力學原理

雙擺是经典物理學中「混沌現象」的代表性模型。不同於單擺的絕對可預測性，雙擺的運動由一組非線性聯立微分方程決定。

本裝置的物理內核採用了經典的 **四階龍格－庫塔法（Runge-Kutta 4th Order, RK4）** 力學數值積分引擎，在每一幀中對拉格朗日方程（Lagrangian Mechanics）進行精確迭代：

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{\theta}_i}\right) - \frac{\partial L}{\partial \theta_i} = 0$$

即使 14 組雙擺的初始釋放角度完全相同，由於第一節擺臂 $L_1$ 存在微小的等比例長度差異，非線性效應會在開局十幾秒後迅速放大（蝴蝶效應）。整排裝置會從最初整齊的機械波浪過渡到極致混亂、翻轉、交錯的混沌之舞。

## 🛠️ 技術架構

- **開發語言**：HTML5 / JavaScript (ES6+)
- **3D 引擎**：[Three.js (r128)](https://threejs.org/)
- **音訊引擎**：瀏覽器原生 Web Audio API (無任何外部音檔，純代碼即時合成正弦波)
- **鏡面反射技術**：CanvasTexture 動態幾何生成 + EquirectangularReflection Mapping 映射
- **視角控制**：Three.js OrbitControls
- **力學引擎**：自研高階四階龍格－庫塔（RK4）非線性數值積分器

## 🚀 快速開始

1. 下載專案中的 `index.html`。
2. 在任意現代瀏覽器（如 Chrome, Safari, Edge）中直接雙擊打開該檔案。
3. 由於瀏覽器的隱私安全規範限制，音訊無法自動播放，請點擊畫面中央的**「開啟 𓃥 白六雙擺律動裝置 (含音效)」**按鈕。
4. 使用手機觸控或電腦滑鼠滑動即可旋轉/縮放觀看視角。點擊左上角小按鈕可展開面板調整引力速度與角度，再次觸碰黑夜畫布即可收合介面，進入純粹的物理放空時空。
